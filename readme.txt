Title: Forecasting Stock Market Opens Using News Sentiment

Description: This project uses Yahoo Finance API to pull realtime stock data and News API to pull realtime news articles. These articles are used classify sentiment on the stock and analyze change in opens. 

Code Structure: 
1. Pull news on stock from last 72 hours
2. Classify sentiment
3. Use Yahoo Finance for stock data
4. Train neural network with both as features
5. Predict stock price for next day open
